## 📂 Estructura del repositorio

| Carpeta | Contenido |
|---|---|
| **01-Machine-Learning** | Regresión, clasificación, clustering, validación cruzada. Proyectos: predicción de precios Airbnb, alquiler de bicicletas, evaluación clínica, simulación de acuario |
| **02-Deep-Learning** | Redes neuronales con Keras/TensorFlow: CNN para clasificación de gestos (dedos), ResNet para clasificación playa/montaña, redes fully connected |
| **03-Big-Data** | Apache Spark (análisis de ventas), MongoDB (consultas desde Python), Neo4j (grafos y Cypher) |
| **04-AI-Models** | Algoritmos genéticos, planificación con PDDL, lógica difusa (control de calefacción), árboles de decisión |
| **05-NLP** | Crawler web, indexador y motor de búsqueda; prácticas de PLN con preprocesamiento de texto y corrección ortográfica |
